PyPaDS-PaDRe: Documentation!
==================================

`PyPaDS-PaDRe`_ is an extension for :ref:`PyPaDS <https://github.com/padre-lab-eu/pypads>`. It introduces common concepts of machines learning, which are not directly related to the base logging functionality of PyPads.
This includes concepts like:

* Splitting / Splitters
* Dataset / Dataset Loaders
* Decisions
* Documentation scanning

.. _PyPaDS-PaDRe: https://github.com/padre-lab-eu/pypads-padre

Install PyPaDS-PaDRe
--------------

PyPaDS-PaDRe is activated automatically if its installed into your environment and you are using PyPaDS. If you install PyPaDS-PaDRe via PIP PyPaDS will be installed too.
